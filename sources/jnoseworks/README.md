#Jnose-suite API

##Event
    1.show
        show state
        "show" is start.
        "shown" is end.

        Sample
        `
        show.ns.*  for example: show.ns.collapse
        shown.ns.*  for example: shown.ns.collapse
        `

    2.hide
        hide state
        "hide" is start.
        "hiden" is end.

        Sample
        `
        hide.ns.*  for example: hide.ns.collapse
        hiden.ns.*  for example: hiden.ns.collapse
        `
